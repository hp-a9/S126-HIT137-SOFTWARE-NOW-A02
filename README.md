# S126-HIT137-SOFTWARE-NOW-A02
A group project that applied file encryption/decryption and a mathematical expression evaluator by means of recursive descent parsing.

### question2.py (Evaluator / Main Controller)

* Reads expressions from the input file
* Calls tokenizer (`q2.py`) and parser (`parser_refactored.py`)
* These two files can be found in the **`Tokenizer` branch**

**Formats:**

* Tokens

* Parse tree

* Final result

* Handles errors gracefully

* Writes output to `output.txt` in the required format

* Returns results as a list of dictionaries
